Завдання 7.1
Створити сторінку, яка містить меню, що дозволяє працювати з поточним вікном браузера. Реалізувати переміщення поточного вікна вверх, вправо, вліво, вниз, збільшення та зменшення його розмірів.

Крім того, ваша головна сторінка повинна змінювати свій фон в залежності від типу браузера. Наприклад, для IE фон сторінки буде синій, для Opera – червоний, а для Firefox – жовтий.
